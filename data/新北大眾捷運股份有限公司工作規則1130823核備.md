# 第一章 總則

# 第 一 條 訂立目的

新北大眾捷運股份有限公司(以下簡稱本公司)為明確規定勞資雙方之權利義務，健全現代經營管理制度，促使勞雇雙方同心協力，並謀事業發展，特依勞動基準法及相關法令訂定本規則。

# 第 二 條 適用範圍

本規則所稱從業人員係指依本公司人事管理規章相關規定僱用從事工作獲致薪資之人員。

# 第二章 受僱與解僱

# 第三條 僱用原則

受僱於本公司之從業人員，均應經審核或甄試合格，方得依規定僱用之。本公司對求職人或所僱用從業人員，不得以種族、階級、語言、思想、宗教、黨派、籍貫、出生地、性別、性傾向、年齡、婚姻、容貌、五官、身心障礙、星座、血型或以往工會會員身分為由，予以歧視；其他法律有明文規定者，從其規定。

# 第四條 報到手續

新進從業人員於接到通知後，應依規定之到職日至本公司親自辦理報到手續，並完成填繳本公司規定表件。

# 第五條 資料變更

從業人員有下列事項之一者，應立即向行政處報備：

一、本人住所變更。

二、姓名、學歷及眷屬之變更。

# 三、其它基於人事管理上公司認為有必要需報備之事項。

# 第六條 臨時勞動契約

本公司得依業務需要及勞動基準法規定，與臨時人員簽訂特定性定期契約。

# 第 七 條 新進試用

本公司得與新進從業人員約定試用，試用期間六個月。於試用期間或試用期滿，考核成績不合格者即停止僱用。試用不合格者將依勞動基準法相關規定辦理，薪資發放至停止僱用日為止。

# 第 八 條 訓練原則

為充實從業人員知識技能，加強專業素質，提高工作效率及服務品質，得依本公司各項訓練計畫辦理教育訓練。

# 第九條 體格檢查

本公司行車人員，必須依「新北市大眾捷運系統行車人員技能體格檢查規則」規定，經本公司指定之醫院辦理體格檢查合格，方能進用，並須依該規則參加專業訓練及技能測驗合格，始得擔任工作。

本公司並將對行車人員之技能、體格及精神狀況，施行定期檢查及臨時檢查，合格者始得續任工作，不合格者應暫停或調整職務，如已喪失工作能力者，依勞工相關法令規定辦理。

# 第十條 技能檢定

本公司捷運系統設施之操作及修護人員，應接受專業技能訓練，經技能檢定合格，始得擔任工作，並定期檢定。

# 第十一條 經預告終止勞動契約

非有下列情事之一者，本公司不得對從業人員預告終止勞動契約：

一、歇業或轉讓時。

二、虧損或業務緊縮、精簡、編併或單位裁撤時。

三、不可抗力暫停工作在一個月以上時。

四、業務性質變更，有減少從業人員之必要，又無適當工作可供

安置時。

五、從業人員對於所擔任之工作確不能勝任時。

# 第十二條 資遣與離職預告

本公司依第11條規定終止勞動契約時，預告期間如下：

一、繼續工作三個月以上一年未滿者，於十日前預告之。

二、繼續工作一年以上三年未滿者，於二十日前預告之。

三、繼續工作三年以上者，於三十日前預告之。

從業人員於接到前項預告後，為另謀工作，得於工作時間請假外出。其請假時數，每星期不得超過二日之工作時間，請假期間之薪資照給。

本公司未依第一項規定期間預告而終止契約時，應發給預告期間之薪資。

從業人員終止勞動契約之預告期間準用第一項規定。

# 第 十三 條 資遣費

凡依第11條規定終止勞動契約之從業人員，除依規定予以預告或未及預告，而依規定發給預告期間之薪資外，並依「新北大眾捷運股份有限公司從業人員退休、資遣及撫卹要點」及勞工退休金條例相關法令發給資遣費。

若從業人員因未符合自願退休條件辭職者，或因違反本規則，或觸犯刑章情節重大經判刑確定而遭解僱(或免職)者，不得申請資遣費之給付。

# 第十四條 不經預告終止勞動契約

凡本公司從業人員有下列情形之一者，本公司得不經預告終止契約：

一、於訂立勞動契約時為虛偽意思表示，使本公司誤信而有受損害之虞者。

二、對於本公司雇主，雇主家屬、雇主代理人或其他共同工作之從業人員及其家屬，實施暴行或有重大侮辱之行為者。

三、受有期徒刑以上刑之宣告確定，而未諭知緩刑或未准易科罰金者。

四、故意損耗機器、工具或其他本公司所有之物品，或故意洩漏本公司技術上、營業上之秘密致本公司受有損害者。

五、無正當理由繼續曠職三日，或一個月內曠職達六日。

六、違反勞動契約或工作規則，情節重大或有以下重大具體事實認定者：

(一)貽誤公務，造成重大過失，導致不良後果。  
(二)要求、期約或收受賄賂；對所經辦業務接受廠商、利害關係人不當招待；挪用、虧欠公款者，經查明屬實。   
(三)圖謀不法利益、言行不檢，致嚴重損害本公司聲譽或重大損失。  
(四)侮辱、誣告或脅迫長官，經查屬實。  
(五)持有、吸食或販售法定毒品及禁藥者。  
(六)攜帶槍砲、彈藥、刀械或依法不得持有之物品至就業場所者。

# 第十五條 懲處解僱

凡受本公司懲處解僱人員，不得再進用為本公司從業人員。

# 第十六條 離職手續

從業人員離職者，應依本公司規定辦妥離職及移交手續。未辦妥離職致本公司遭受損失，本公司得要求賠償，情節重大者依相關司法程序辦理。

# 第十七條 服務證明書

勞動契約終止時，經從業人員之請求，本公司應發給服務證明書。

# 第 十八 條 調動原則

調動人員之工作，不得違反勞動契約之約定，並應符合下列原則：

一、本公司因企業經營上需要，於不違背勞動契約，且對從業人員薪資及其他勞動條件不作不利之變更，並考量從業人員及其家庭生活利益，得依從業人員之體能及技術調整從業人員之職務或工作地點，其年資合併計算；從業人員有正當理由時，得申請覆議。

二、從業人員之調動工作地點過遠，本公司應予以必要之協助。

# 第十九條 留職停薪

本公司從業人員具有下列情形之一者得申請留職停薪，經公司核准並辦妥移交手續後始得生效，留職停薪期間年資不計：

一、從業人員服務滿六個月，於子女未滿三歲前（包含擬收養子女先期共同生活之期間）需育嬰者，但不得逾二年。同時撫育子女二人以上者，其育嬰留職停薪期間應合併計算，最長以最幼子女受撫育二年為限。  
二、應徵入伍服役者。  
三、其他因特殊情形呈請核准者。

# 第三章 薪資、津貼及加班費用

# 第二十條 薪資之議定

從業人員之薪資由本公司與從業人員議定之。但從業人員在正常工作時間內所得之報酬，不得低於基本工資。

# 第二十一條 薪資計算

採月薪制從業人員之計時及計日薪資計算方式如下：

一、計日薪資之計算方式為月薪薪資除以三十日。  
二、計時薪資之計算方式為月薪薪資除以兩百四十小時(三十日乘以八小時)。

前項月薪薪資包含基本薪、職務加給、專業加給。

# 第二十二條 薪資發放時間

薪資發放時間，依下列規定辦理：

一、從業人員之薪資，均採按月預付，並於每月十日發給為原則（如遇假日則提早一日發放）。離職者應繳回離職生效日起溢領之薪資；若當月新進從業人員因作業不及無法於當月發放薪資，將於次月份補發。  
二、從業人員薪資給付，應以法定通用貨幣為之，並得將之劃撥至從業人員在本公司指定之金融機構所開列之帳戶，除法令另有規定或與從業人員另有約定外，全額直接給付從業人員，並應提供工資各項目計算方式明細。

# 第二十三條 延長工時之薪資加給標準

正常工作日延長工時(每日工作時間超過八小時之部分) 薪資發給：

一、延長工作時間在二小時以內者，按平日每小時薪資額加給三分之一以上。  
二、再延長工作時間在二小時以內者，按平日每小時薪資額加給三分之二以上。  
三、延長之工作時間連同正常工作時間，一日不得超過十二小時；延長之工作時間，一個月不得超過四十六小時。

休息日工作之薪資發給：

一、因業務需要，本公司經從業人員同意於休息日工作者，工作時間在二小時以內者，其工資按平日每小時薪資額另再加給一又三分之一以上；工作二小時後再繼續工作者，按平日每小時薪資額另再加給一又三分之二以上。逾八小時部分，按平日每小時薪資額另再加給二又三分之二。本公司因天災、事變或突發事件之發生，有使從業人員於休息日工作之必要者，出勤工資計算方式，依勞動基準法第24條第2項規定計給。

二、休息日工作之時間，須計入延長之工作時間總數。

休假日(國定假日)工作之薪資發給：

一、主管因業務需要並經從業人員同意，指定從業人員出勤時或為自行出勤並報核准者，不論(正常班人員以凌晨零時為起算點，輪班人員以班表下班時間後滿七小時為起算點)，均加發一日薪資或選擇補休，如有延長工時者，依延長工時規定申報加班費或選擇補休(兩者擇一)。

二、延長工時之時間，須計入延長之工作時間總數。

因事變、或突發事件時之薪資發給：

一、本公司要求從業人員於正常工作時間以外工作者，超時部份之延長工時薪資加給二倍，並可依延長工時時數(連續工作時數，未滿四小時以半日計，逾四小時未滿八小時以一日計，依此類推)給予補休，並應適當休息十二小時後方可再出勤。

二、如需停止從業人員之例、休假時，於原訂正常工時內之工時，不論出勤時數，均加發一日薪資，超時部份之延長工時薪資加給二倍，並給予補休一日或選擇發給一日薪資(如有超時者可再依前列標準補休)，並應適當休息十二小時後方可再出勤。

天然災害發生時之薪資發給：

一、因本公司係屬交通公用事業單位，除安全有虞外，輪班人員、輪值災害應變中心及其他維持營運所必需之人員，仍應出勤。若因政府宣布「停止辦公日期間」，原應上班且出勤者之薪資以停止辦公期間之實際出勤時數按平日每小時薪資額加倍發給(原有薪資外再加一倍)並按實際值勤時數給予補休，惟同一班次除加班費外，最多僅得補休十二小時；停止休假日出勤者依上述規定辦理，所加給加班費及補休時數最少應以八小時計算。

二、另因天災並於停止上班期間延長工作時間者之薪資，依實際延長工作時數按平日每小時薪資額加倍(乘以二)發給之，並給予補休。

三、前二目情形，同一事件之出勤者應適當休息十二小時後方可再出勤。

四、第一目所稱「停止辦公日期間」係指天然災害之範圍達到行政院訂定之「天然災害停止上班及上課作業辦法」中停止辦公之標準，經政府機關宣布停止辦公時；若依「天然災害停止上班及上課作業辦法」及參酌天災侵襲實況已達停止辦公標準，惟因屬假日致政府未予宣布停止辦公者，由行政處事後簽報總經理予以認定。

前述出勤時數得申報加班費或選擇補休。

# 第二十四條 加班申報

加班申報應填具加班申請單，並填寫加班事由、申請起迄時間與申請時數或選擇補休(二者擇一申請)，呈權責主管審核；為實際控管考勤，未經核准者不予計算。加班費之最小計算單位為一小時。

# 第二十五條 補休

選擇補休時，依下列規定辦理：

一、延長工作時數之選擇補休時，最終補休期限不得超過特休年

度。

二、每次補休請假得以時計。

三、補休期限屆期或契約終止未補休之時數，應依延長工作時間或休息日工作當日之薪資計算標準發給薪資。

四、補休應依從業人員延長工作時間或休息日工作事實發生時間先後順序補休。

# 第二十六條 延長工時或假日出勤之薪資發放

延長工時或假日出勤之薪資，應於申請資料送達人事單位後一個月內發給；如從業人員於發放前離職或因故終止勞動契約，則於離職或終止勞動契約後一個月內結算發放。

# 第四章 工作時間、休息、休假、請假

# 第二十七條 工作時間

正常班從業人員正常工作時間每日不超過八小時，每週不得超過四十小時。倘有正常工作時間以外工作之必要者，經勞資會議同意後，從業人員同意本公司各單位得因業務需要，要求從業人員於正常工作時間以外工作(按：即加班，以下亦同)，並依本公司加班相關規定程序申請；其延長工作時間，連同正常工作時間，一日不得超過十二小時。

應在本公司規定之時間或排定之班表上班，但有特殊情事經授權主管裁決核可者，不在此限。

從業人員於工作時間內，因本公司事務必須外出時，應事前向所屬主管報准後始得外出，並填具公出、出差申請單。

子女未滿二歲須從業人員親自哺(集)乳者，除規定之休息時間外，本公司將每日另給哺(集)乳時間六十分鐘；每日正常工作時間以外之延長工作時間達一小時以上者，另給予哺(集)乳時間三十分鐘。哺(集)乳時間，視為工作時間。

從業人員為撫育未滿三歲子女，得請求下列所定事項之一：

一、每天減少工作時間一小時；減少之工作時間，不得請求報酬。  
二、調整工作時間。

從業人員為前二項哺(集)乳時間、減少或調整工時之請求時，本公司不得拒絕或視為缺勤而影響其全勤獎金、考績或為其他不利之處分。

# 第二十八條 女性從業人員夜間工作

女性從業人員輪班之工作時間在午後十時至翌晨六時之時間內者，本公司提供必要之安全衛生設施，並供給夜宿宿舍或交通工具接送。但妊娠或哺乳期間之女性從業人員在午後十時至翌晨六時止，仍不輪排工作時間。

# 第二十九條 休息時間

從業人員每繼續工作四小時，至少應有三十分鐘之休息，其休息時間長短依工作性質調配。若因實行輪班制或其工作有連續性或緊急性者，得在工作時間內另行調配其休息時間。

# 第三十條 例假及休息日

從業人員每七日中應有二日之休息，其中一日為例假，一日為休息日，薪資照給。

# 第三十一條 休假日(國定假日)

內政部所定應放假之紀念日、節日、勞動節及其他中央主管機關指定應放假之日，均應休假，薪資照給。

# 第三十二條 特別休假之日數及排定方式

從業人員於本公司繼續工作滿一定期間者，本公司應依下列規定給予特別休假：

一、六個月以上一年未滿者，三日。

二、一年以上二年未滿者，七日。

三、二年以上三年未滿者，十日。

四、三年以上五年未滿者，每年十四日。

五、五年以上十年未滿者，每年十五日。

六、十年以上者，每一年加給一日，加至三十日為止。

前項從業人員之工作年資自受僱當日起算。

特別休假期日，由從業人員排定之，每次請假應至少半日；本公司應於從業人員符合前項所定特別休假條件之日起三十日內，告知從業人員排定特別休假。

# 第三十三條 未休畢特別休假薪資發給及書面通知

從業人員之特別休假，因年度終結或契約終止而未休之日數，本公司應發給薪資。

從業人員每年特別休假期日及未休假之日數所發給之薪資數額，由本公司記載於薪資清冊，並於給付未休假薪資時，一併通知從業人員。

# 第三十四條 投票

於各類公職人員選舉(罷免)投票日之給假規定，依下列規定辦理：

一、周休二日或原休息日舉行投票，不另給假。

二、輪班人員之休班(R班，視同周休二日)或原休息日舉行投票，不另給假。

三、屬「政府行政機關辦公日曆表」非假日之工作日舉行投票，正常班人員均放假一日，輪班人員當月減計一工作班。

四、投票日輪班人員皆應依班表所定時間準時到勤，惟得調整班表或給予適當公假時間，以不妨礙其投票。投票當日出勤人員屬休假日出勤，一律加給薪資。

五、各項公職人員選舉以當事人具選舉罷免權為準，不論中央或地方選舉均同。

# 第三十五條 給假規定

從業人員因婚、喪、疾病或其他正當理由得請假，假別分為婚假、事假、普通傷病假、生理假、喪假、公傷病假、產假、產檢假、陪產檢及陪產假、家庭照顧假、捐贈器官假、原住民族歲時祭儀及公假等十三種。准假日數及薪資給付如下：

一、婚假：從業人員結婚者給予婚假十四日(不含例假日)，薪資照給。除因特殊事由經副總經理以上主管核准延後給假者外，應自結婚之日前十日起三個月內請畢。請假時須檢附結婚登記之戶籍謄本。

二、事假：從業人員因事必須親自處理者，得請事假；一年內合計給予事假十四日，未超過五日部分，薪資照給。連續請假二日以上者，本公司得要求從業人員提出相關證明文件。

三、普通傷病假：從業人員因普通傷害、疾病或生理原因必須治療或休養者，得依下列規定請普通傷病假，請假連續二日(含)以上者，須附繳醫療證明。(一年內合計未超過十日部分，薪資照給；一年內合計十一日以上，未超過三十日部分，薪資折半發給，超過三十日不給薪。)

(一)未住院者，一年內合計不得超過三十日。

(二)住院者，二年內合計不得超過一年。

(三)未住院傷病假與住院傷病假，二年內合計不得超過一年。經醫師診斷，罹患癌症(含原位癌)採門診方式治療或懷孕期間需安胎休養者，其治療或休養期間，併入住院傷病假計算。普通傷病假超過前款規定之期限，經以事假或特別休假抵充後仍未痊癒者，經本公司同意得予留職停薪以一年為限。逾期未癒者得予資遣。

四、生理假：女性從業人員因生理日致工作有困難者，每月得請生理假一日，全年請假日數未逾三日，不併入病假計算，其餘超過日數併入病假計算；生理假連同普通傷病假一年內未超過十日部分，薪資照給，一年內合計十一日以上，未超過三十日部分，薪資折半發給，超過三十日不給薪。

五、喪假：薪資照給。從業人員喪假得依習俗於百日內分次申請。

(一)父母、養父母、配偶喪亡者，給予喪假十五日。

(二)繼父母、配偶之父母、子女喪亡者，給予喪假十日。

(三)曾祖父母、祖父母、配偶之繼父母、養父母喪亡者，給予喪假六日。

(四)兄弟姊妹喪亡者，給予喪假五日。

(五)配偶之祖父母死亡者，給予喪假三日。

除繼父母、配偶之繼父母，以從業人員或其配偶於成年前受該繼父母扶養或於該繼父母死亡前仍與共居者為限外，其餘喪假應以原因發生時所存在之天然血親或擬制血親為限。喪假得分次申請，每次不得少於半日，並應於死亡之日起百日內請畢，並需檢附死亡證明書或戶籍謄本等有關文件以證明親屬關係。

六、公傷病假：依「新北大眾捷運股份有限公司從業人員退休、

資遣及撫卹要點」辦理。

# 七、產假：

(一)分娩者，給予產假八星期。  
(二)懷孕滿五個月以上流產者，給予產假六星期。  
(三)懷孕三個月以上未滿五個月流產者，給予產假四星期。  
(四)懷孕未滿三個月流產者，給予產假二星期。  
(五)前述規定之女性從業人員受僱工作在六個月以上者，停止工作期間薪資照給，未滿六個月者減半發給。

(六)女性從業人員請產假須提出證明文件；流產應附健保特約醫療機構開立之醫生診斷證明書。各產假日數均含休假日，並最晚於分娩或流產當日起算，並應一次請畢。

八、產檢假：女性從業人員妊娠期間，給予產檢假八日。產檢假期間，薪資照給(須檢附健保特約醫療機構或醫師證明書，並不得保留至分娩後)。

九、陪產檢及陪產假：從業人員陪伴其配偶妊娠產檢或其配偶分娩時，給予陪產檢及陪產假七日。除陪產檢於配偶妊娠請假外，從業人員陪產之請假，應於配偶分娩之當日及其前後合計十五日期間內為之。陪產檢及陪產假期間，薪資照給，並須檢附配偶生產證明。

十、家庭照顧假：從業人員於其家庭成員預防接種、發生嚴重之疾病或其他重大事故須親自照顧時，得請家庭照顧假；其請假日數併入事假計算，全年以七日為限。家庭照顧假薪資之計算，依事假規定辦理。

十一、捐贈器官假：捐贈骨髓或其他器官依實際需要給假(同一事由，需一次請畢)，薪資照給。應檢具合法醫療機構或醫師證明書。

十二、原住民族歲時祭儀：各原住民族從業人員遇所屬原住民族「原住民族歲時祭儀」時，放假一日，薪資照給，放假日期以行政院原住民族委員會公告為準。得持戶籍謄本或戶口名簿等足資證明其族別之文件，向人事單位辦理放假。

十三、公假：從業人員有下列各款情事之一者，需檢附相關證明文件，依實際需要日數給予公假，薪資照給：

(一)奉派參加政府召集之集會。  
(二)參加政府舉辦與職務有關之考試，經單位主管核准者。

(三)依法受各種兵役召集。

(四)參加政府依法主辦之各項投票。

(五)因執行職務或上下班途中發生危險以致傷病，必須休養或療治。

(六)奉派或奉准參加與其職務有關之訓練進修，其期間在一年以內者。

(七)奉派考察或參加國際會議。

(八)應國內外機關團體邀請，參加與其職務有關之各項會議或活動，或基於法定義務出席作證、答辯，經單位主管核准者。

(九)參加本公司舉辦之活動，經單位主管核准者。

(十)因法定傳染病經各級衛生主管機關認定應強制隔離者。但因可歸責於當事人事由而罹病者，不在此限。

從業人員申請生理假、產假、產檢假、陪產檢及陪產假、家庭照顧假、育嬰留職停薪、安胎休養請假時，本公司不得拒絕或視為缺勤而影響其全勤獎金、考績或為其他不利之處分。

事假、普通傷病假、生理假、產檢假、家庭照顧假，得以時計。喪假每次請假應至少半日。陪產檢及陪產假，可依員工實際需求選擇以半日或小時為請假單位，擇定後不得變更。婚假、捐贈器官假，每次請假應至少一日。

計算週期均自每年一月一日起至同年十二月三十一日止。

有本規則第 12 條及 16 條情事者，事假及普通傷病假涉給薪部分，依前揭週期實際在職日數核給，並於最後工作日核算費用。

# 第三十六條 請假手續

本公司從業人員須請假、公假或休假時，應事先申請，經核准後方可離開工作崗位。但有急病或緊急事故，得由其同事或家屬親友代辦或補辦請假手續。

如遇急病或臨時重大事故，應立即委託同事、家屬或親友代辦手續，(但特殊事故，經單位主管事前核准補辦請假手續者不在此限)，如需補述理由或提供證明，應於三日內提送工作單位按權責

核准。

請假、公假或休假人員職務，應委託同事代理。前項在假人員，應將經辦事項確實交代代理人。

本公司為貫徹休假制度，經單位排定之特別休假者，從業人員應確實遵守及配合辦理前項之手續。

# 第三十七條 代刷卡

從業人員均應親自刷卡或簽到(退)，經查明託人代刷卡或簽到(退)者記過一次，實際缺勤時數以曠職論；代人刷卡或簽到(退)者記大過一次。

# 第三十八條 曚職

未辦請假、公假或休假手續而擅離職守或假期已滿仍未銷假或請假有虛偽情事者，查明屬實以曠職論。曠職以時計算，累積滿八小時以一日計；其與曠職期間連續之例假日應予扣除，並視為繼續曠職。

本公司於事前徵得從業人員同意於休假日工作後，從業人員即有在休假日出勤之義務，如從業人員屆時未到工，除事先告知主管或有正當理由者外，本公司得予以曠職處分。

# 第五章 退休

# 第三十九條 退休年齡之認定

從業人員退休年齡之認定以戶籍記載為準，自出生之日起足歲計算。

# 第四十條 自請退休

從業人員有下列情形之一者，得自請退休，並須於一個月前提出申請，經本公司總經理核准後退休：

一、工作十五年以上年滿五十五歲者。  
二、工作二十五年以上者。  
三、工作十年以上年滿六十歲者。

# 第四十一條 強制退休

從業人員有下列情形之一者，本公司得強制其退休：

一、年滿六十五歲者。  
二、身心障礙不堪勝任工作者。

前項第一款所規定之年齡，對於擔任具有危險、堅強體力等特殊性質之工作者，本公司得報請中央主管機關核准調整，但不得少於五十五歲。

# 第四十二條 退休金之提撥

從業人員退休金給與標準如下：

一、本公司按月提繳勞工退休金，儲存於勞保局設立之勞工退休金個人專戶。勞工退休金自勞工到職之日起提繳至離職當日止。  
二、本公司每月負擔之勞工退休金提繳率，不得低於勞工每月薪資百分之六。

# 第四十三條 退休金請求

從業人員適用勞動基準法退休金規定者，其請領退休金權利，自退休之次月起，因五年間不行使而消滅。請領退休金權利不得讓與、抵銷、扣押或供擔保。

前項請領之退休金，從業人員得檢具證明文件，於金融機構開立專戶，專供存入勞工退休金之用。專戶內之存款，不得作為抵銷、扣押、供擔保或強制執行之標的。

# 第六章 應遵守之紀律

# 第四十四條 工作守則

一、遵守公司一切規章、通知、公告及作業程序，盡忠職守，服從主管之命令、指揮及監督。  
二、應遵守本公司之經營理念，塑造本公司良好企業形象。  
三、應以親切、和善與有禮貌之工作態度，提供民眾高品質之大眾捷運運輸服務。  
四、應公私分明、謹言慎行、操守廉潔、摒除一切不良習慣，同事間要和睦相處、相互尊重、維護公司榮譽。

五、應堅守崗位，分層負責，並與有關部門保持密切聯繫，同心協力完成經辦事項，以利公司業務之推行。

六、因業務上之需要，經主管指派之工作，應服從命令，聽從工作指派，戮力工作，提昇工作品質及增進經營績效，不得藉故推諉。

七、應隨時注意工作場所之整潔美觀及安全，未經核准不得私帶親友進入工作場所，不得私帶公物離開工作場所。

八、對顧客或來賓應保持謙和有禮與友善，不得有輕妄、傲慢、敷衍或相應不理之態度。

九、從業人員應愛護本公司之一切設施，節省物資，保持良好之工作環境，增進工作效率及技術，以求公司最大利益，如有故意毀損，除依民法賠償外，並依規定懲處。

十、從業人員不得探詢他人之薪資資料。

十一、從業人員於工作場所，不得有得賭博、酗酒、毆鬥、吵鬧、嬉戲、赤膊、穿拖鞋、木屐或吃檳榔等，並除指定地區外，禁止吸菸，或有違反大眾捷運法第四十八條至第五十條之一之行為。

十二、對於業務或職務上之機密文件圖表，不得對外洩漏及遺失。

# 第四十五條 輪班制度

從業人員應遵守下列輪班制度：

一、輪班從業人員其工作班次，每週更換一次。但經從業人員同意者不在此限。

二、更換班次時，至少應有連續十一小時之休息時間。

三、輪班人員每月實際應工作總時數依正常班人員每月工作總時數計算(因天災或事變而減少之工時不予扣減)，休息時間另為安排。班次按月合計，加計當月國定假日為休假日(減少當月應上之班數)。如有法令或案例或工作性質特殊，經核准縮短或延長工時有案者，工作時數之計算從其規定。

四、每月上班超過當月應上班(時)次者，超過之班次以假日出勤核計。

五、輪班人員當月應工作班數有不足時，得以超過八小時換算一工作班抵充之，若定期結算仍有超時，方可以延長工時核發延時薪資。連續月份之上班時數，累積計算，當月如有不足班次，得於次月之超領班次抵充。

六、為配合營運需要，於輪班時段內，無法安排固定用餐及休息時間，且未再安排人員替代工作者或需隨時待命者，其規定到勤時間至離勤時，皆視為工作時間。

七、從業人員因故需與他人換班時，應提出調換班申請，經核准後，始完成調換班程序。

八、輪班交接時，應將事物交待清楚，以明責任。

# 第七章 考核、獎懲、陞遷

第四十六條 考核

本公司為激勵士氣，確保工作精進，得視需要辦理年度從業人員考核(績)。

定期及臨時人員除於僱用契約規定者外，不另辦理平時或年終考核。

第四十七條 獎懲及升遷

本公司為激勵士氣，確保工作精進，得視從業人員表現辦理獎懲及升遷。

# 第八章 職業災害補償及撫卹

第四十八條 職業災害補償

本公司從業人員退休、資遣、撫卹及職業災害補償之請領依「新北大眾捷運股份有限公司從業人員退休、資遣及撫卹要點」辦理。

第四十九條 非職業災害撫卹

本公司從業人員退休、資遣、撫卹及職業災害補償之請領依「新北大眾捷運股份有限公司從業人員退休、資遣及撫卹要點」辦理。

第五十條 職業災害補償請求時效

因職業災害之受領補償權，自得受領之日起，因二年間不行使而消滅。

受領補償之權利，不因從業人員離職而受影響，且不得讓與、抵

銷、扣押或擔保。

勞工或其遺屬依本法規定受領職業災害補償金者，得檢具證明文件，於金融機構開立專戶，專供存入職業災害補償金之用。

前項專戶內之存款，不得作為抵銷、扣押、供擔保或強制執行之標的。

# 第九章 社會保險、福利措施與安全衛生

第五十一條 勞工保險、就業保險、勞工職業災害保險及全民健康保險

從業人員均由公司依法令規定辦理勞工保險、就業保險、勞工職業災害保險及全民健康保險，並依相關法令享有保險給付權利。對於同仁生育、傷病、失能、老年、死亡等之給付，亦由公司依「勞工保險條例」、「全民健康保險法」及「勞工職業災害保險及保護法」等規定，由公司辦理轉請勞工保險局及全民健康保險局給付。

第五十二條 從業人員福利

本公司依職工福利金條例設置職工福利委員會，提撥職工福利金，辦理福利事項。職工福利金之保管動用，由本公司職工福利委員會負責辦理。

第五十三條 夜點費

請領夜點費之時段標準為輪值小夜班(午班)人員，下班時間於二十二時以後(含)者，或輪值大夜班人員，二十二時以後(含)在勤者，皆可支領夜點費，並依本公司每年營收狀況提勞資會議檢討調整。

第五十四條 勞工教育

本公司為預防職業災害，保障從業人員安全與健康，依法辦理勞工教育，並依法令規定設置安全衛生管理單位及職業安全衛生管理人員，辦理職業安全衛生事宜。

第五十五條 安全衛生

本公司依職業安全衛生法規規定，規範之職業安全衛生工作及事項，從業人員應遵照相關規定配合辦理。

# 第十章 其他

# 第五十六條 勞資會議

本公司為協調勞資關係，增進彼此瞭解，促進勞資合作，提高工作效率，依「勞資會議實施辦法」舉辦勞資會議。定期開會並三個月至少召開一次為原則，相互溝通意見，勞雇雙方應本和諧誠信原則，協商解決問題。

# 第五十七條 從業人員申訴性騷擾處理制度

本公司為防治性騷擾行為並維護性別工作平等及人格尊嚴，特依據性別工作平等法規定訂定「新北大眾捷運股份有限公司性騷擾防治措施、申訴及懲戒要點」，從業人員應遵照相關規定配合辦理。

# 第五十八條 申訴制度

從業人員於本公司內如有關於本身之合法權益遭受侵害、不當處置或在執行業務上遭到困難，經反映仍無法獲得合理解決時，得提出申訴。

從業人員如有不循正當途徑表達意見或提出建議者，其行為若已構成「任意謾罵」、「藉端聚眾、非法煽動風潮」、「虛構或傳播不妥言論，足以影響人心」等事實，本公司將依本規則及相關規定辦理。

# 第五十九條 補充規定

本規則若有法令修改、未盡事宜或涉及從業人員其他權利義務事項，本公司得視實際需要，按照有關法令規定辦理之。

# 第六十條 實施

本規則經公告後施行，修正時亦同。
